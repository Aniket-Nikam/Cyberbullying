Firebase Rules Setup

Your account needs admin role set in the database, and the correct security rules applied. Follow these steps:

1

Go to Firebase Console → Realtime Database → Rules

Open Firebase Console and click the Rules tab.

2

Set your UID as admin in the database

Go to Data tab, find or create users / YOUR_UID / role and set its value to "admin". Your UID:

Sign in first to see your UID
3

Paste these rules and click Publish

Copy the rules from the firebase-rules.json file and paste into the Rules editor, then click Publish.

4

Click Refresh in the admin panel

Once rules are published and your role is set, click the Refresh button in the top bar to reload all data.
